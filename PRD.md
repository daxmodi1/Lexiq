# LEXIQ — Product Requirements Document
> AI-Powered Dictionary & Vocabulary Builder | v1.0 | Status: Draft | Phase: MVP

## 1. Product Overview

Lexiq is an AI-powered dictionary and personal vocabulary builder that helps users discover, understand, and truly learn words. Unlike static dictionaries, Lexiq builds an intelligent vocabulary profile for each user, tracks their mastery over time, and delivers personalized daily practice through quizzes and challenges.

| Attribute | Details |
|---|---|
| Product Name | Lexiq |
| Version | 1.0 (MVP) |
| Target Users | Students, professionals, language learners, readers |
| Primary Goal | Help users build a lasting personal vocabulary through AI and spaced practice |
| Tech Stack | Next.js (latest), Tailwind CSS, PostgreSQL, Prisma, Groq API, Clerk |

## 3. Feature Specifications

### 3.1 Word Search & Auto-Add

When a user types a word, the Groq API (LLaMA 3.3 70b) generates a comprehensive dictionary entry. The user can then add the word to their personal vocabulary list manually or automatically.

#### Word Entry Card Fields

| Field | Description |
|---|---|
| Word | The word itself, displayed prominently |
| Phonetic / Pronunciation | IPA notation + audio button (Web Speech API) |
| Part of Speech | Noun, verb, adjective, etc. |
| Definition(s) | Primary and secondary definitions |
| Example Sentences | 2–3 contextual examples generated by AI |
| Synonyms | Related words with similar meanings |
| Antonyms | Opposite meaning words |
| Etymology | Word origin, root language, historical usage |
| Word Family | Related forms: decide → decision, decisive, indecisive |
| Mnemonic | AI-generated memorable trick to recall the word |
| Difficulty Tag | Auto-classified: Beginner / Intermediate / Advanced / Rare |
| Mastery Score | User-specific score (0–100), initialized at 50 on add |
| Date Added | Timestamp when word was added to user vocabulary |

#### Auto-Add Toggle Behavior

| Mode | Behavior |
|---|---|
| Auto-Add ON | Word is added to vocabulary list immediately when searched |
| Auto-Add OFF | After generating the entry, a prompt appears: "Add this word to your list?" with Accept / Skip buttons |

### 3.2 Word Mastery Score System

Each word in a user's vocabulary list has a Mastery Score between 0 and 100. This score reflects how well the user knows the word based on their quiz performance. The system uses a modified spaced repetition approach — but decay is conditional, not automatic.

#### Score Initialization

- When a word is added to the vocabulary list, its Mastery Score starts at **50**
- This neutral starting point reflects "seen but not yet tested"

#### Score Changes — Quiz Performance

| Event | Score Change | Notes |
|---|---|---|
| Correct answer (first try) | +8 | Full credit |
| Correct answer (second try) | +4 | Partial credit |
| Wrong answer | -10 | Also triggers 30-day queue reset |
| Skipped / Timed out | -3 | Soft penalty |
| Perfect streak (5 in a row) | +5 bonus | Applied on top of correct answer bonus |

> **Important:** Score is always clamped between 0 and 100. It will never go below 0 or above 100 regardless of streaks or penalties.

#### 30-Day Queue System (Decay Trigger)

Score does **NOT** decay automatically on a timer. Decay only occurs when **both** conditions are met:
1. The word has not been reviewed in **30+ days**, AND
2. The user answers it **wrong** when it reappears

If either condition is missing — no decay occurs.

**How the 30-day queue works:**

- Every word tracks a `last_reviewed_at` timestamp
- A background job runs nightly and finds words where `last_reviewed_at` is older than 30 days
- These words are pushed into the user's **Re-Review Queue**
- The Re-Review Queue feeds into daily practice sessions with higher priority
- If the user answers a re-queued word **correctly** — score stays as-is, word exits the queue
- If the user answers a re-queued word **wrong** — score drops by 10 (same as regular wrong answer)
- The word stays in the Re-Review Queue until answered correctly at least once

#### Mastery Score Tiers (UI Display)

| Score Range | Tier Label | Badge Color |
|---|---|---|
| 0 – 29 | Learning | Red |
| 30 – 59 | Familiar | Orange |
| 60 – 79 | Practiced | Blue |
| 80 – 94 | Proficient | Green |
| 95 – 100 | Mastered | Purple |

### 3.3 Difficulty Tags

When a word is looked up, the Groq API automatically assigns a difficulty tag based on word frequency, complexity, and common usage context.

| Tag | Criteria |
|---|---|
| Beginner | Top 5,000 most common English words. Simple everyday vocabulary. |
| Intermediate | Words commonly encountered in academic or professional settings. |
| Advanced | Low-frequency words, discipline-specific, or literary vocabulary. |
| Rare | Archaic, highly technical, or obscure words rarely used in modern writing. |

Difficulty tags are used to: filter vocabulary lists, balance daily quiz question difficulty, and provide stats (e.g., "You have 12 Advanced words — keep going!").

### 3.4 Daily Questions

There are two types of daily questions. Both reset at midnight UTC.

#### Type 1: Global Word of the Day

- One word is selected each day and shown to **all users** on the platform
- Selection is curated — the system picks a random Intermediate or Advanced word from a global word bank (not the user's personal list)
- All users see the same word, creating a shared daily challenge
- Format: Multiple choice — "What does [word] mean?" with 4 options
- Answering correctly contributes to the user's **Word of the Day Streak**
- Streak count is displayed on the user's profile and dashboard

#### Word of the Day Streak

- Increments by 1 each day the user correctly answers the Global Word of the Day
- Breaks if the user misses a day or answers wrong
- Milestones at 7, 30, 60, 100 days trigger achievement badges
- A grace period of 24 hours is applied — streak does not break until 48h of inactivity

#### Type 2: Personal Vocabulary Quiz

- Personalized to each user's own vocabulary list
- 3 fill-in-the-blank questions served daily
- Words are selected based on: lowest mastery score first, then words in the re-review queue, then random from the list
- Question format: A sentence with the target word blanked out — user types the word
- AI grades the answer — accepts spelling variations within reason

**Word Selection Priority:**

| Priority | Source | Condition |
|---|---|---|
| 1st (Highest) | Re-Review Queue | Words older than 30 days that were answered wrong |
| 2nd | Low Mastery Words | Score below 40 |
| 3rd | Medium Mastery | Score 40–69, not reviewed in 7+ days |
| 4th (Fallback) | Random | Any word in the user's list |

---

### 3.5 Quiz & Practice Modes

#### Timed Challenge Mode

- User selects a difficulty filter (Beginner / Intermediate / Advanced / All)
- 60-second countdown timer
- Words are shown one by one — user selects the correct definition from 4 options
- Score = number of correct answers in 60 seconds
- Best score is saved to user profile
- Mastery scores update based on performance in timed challenge

#### Practice Modes Overview

| Mode | Description | Status |
|---|---|---|
| Fill in the Blank | Sentence with word removed, user types the word | MVP |
| Definition Match | Match word to correct definition from 4 options | MVP |
| Timed Challenge | 60-second rapid-fire multiple choice | MVP |
| Use in a Sentence | User writes a sentence; AI grades for correctness | Post-MVP |
| Reverse Quiz | Show definition, guess the word | Post-MVP |

---

### 3.6 Vocabulary Collections

Users can organize their words into named collections (e.g., "GRE Prep", "Medical Terms", "Words from Dune").

- A word can belong to multiple collections
- Collections can be filtered in practice modes
- Default collection: "All Words"
